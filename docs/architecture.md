# Cognitive Physics Engine

The core innovation behind AXIOMX's UI is its **Cognitive Physics Engine**. Traditional AI platforms render thoughts as static chat logs. AXIOMX renders thoughts as a **living, physical topology** that responds to the emotional and logical state of the AI agents.

## 1. Dynamic Graph Mutation & Cognitive Gravity
The network reorganizes itself in real-time based on the reasoning state:
- **Gravitational Anchors:** High-confidence logic chains act as gravitational centers, pulling related thoughts inward.
- **Centrifugal Drift:** Low-confidence or highly speculative branches drift to the periphery of the graph.
- **Spring/Repulsion Physics:** The graph fluidly re-balances on every frame using a custom d3-inspired physics simulation running via `requestAnimationFrame`.

## 2. Thought Collision Systems
When agents disagree (e.g., the Critic aggressively challenges the Researcher), the engine injects literal turbulence into the graph:
- **Contradiction Shockwaves:** Nodes emit red, pulsating ripples that push adjacent nodes away.
- **Turbulence Injection:** The physics simulation receives random velocity spikes, making the graph physically "shake" as the AI debate intensifies.
- **Edge Fragmentation:** Connections between disagreeing agents become dashed and glitchy, representing logical instability.

## 3. Holographic Depth & Telemetry
Every node is more than a text box. It is a telemetry endpoint:
- **Live Stream Injection:** As GPT-4o generates tokens, the nodes expand and re-balance dynamically.
- **Ambient Glow:** Each node casts a radial glow corresponding to its agent's color (e.g., Cyan for Planner, Crimson for Risk Analyst).
- **Z-Axis Elevation:** Important thoughts scale up and cast deeper drop shadows, creating a 3D holographic effect.
